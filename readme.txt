based on http://reangdblog.blogspot.com/2016/06/emacs-ide-go.html

C-c C-j - перейти к реализации функции под курсором (вернуться назад, можно через M-,)
C-c C-f i - перейти к секции "import" текущего файла
C-c C-f f - перейти к началу функции, внутри которой находится курсор
C-c C-f a - перейти к аргументам текущей функции
C-c C-f d - перейти к комментариям функции
C-c C-f r - перейти к описанию возвращаемого значения для функции
C-M-a - перейти к началу функции
C-M-e - перейти к концу функции
M-s M-s - toggle speedbar
C-c C-a - annotate region
M-s r - start newsticker
C-c t - prefix for comment-tags
